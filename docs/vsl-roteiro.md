# Roteiro VSL — Shark 100% Green
**Versão:** 1.0 — Maio/2026
**Duração total estimada:** 10–14 minutos
**Formato:** Talking head + screencap/screenshare + B-roll de prints
**Idioma:** Português Brasileiro
**Onde é exibido:** Página `/obrigado` (campo `VSL_EMBED_URL`)

---

## Instruções gerais de gravação

- Fale olhando diretamente pra câmera — tom de conversa, não de apresentação corporativa.
- Sem fundo genérico: use setup dark com iluminação neon verde ou fundo preto simples. Reforça a identidade visual.
- Sempre que o roteiro indicar `[TELA]`, corte pra screencap ou gravação de tela. Depois volta pro talking head.
- Não use música alta em cima da voz. Trilha ambiente leve (lo-fi dark) no fundo, mixada baixo.
- Corte os vícios de linguagem na edição (ãã, né, tipo). O script já foi escrito sem eles pra facilitar.

## Notas de produção

| Item | Recomendação |
|---|---|
| Duração total | 10–14 min (não cortar demais, não inflar) |
| Resolução mínima | 1080p / 30fps |
| Thumbnail do vídeo | Rosto + texto impactante em fundo escuro (ex: "Como funciona o método") |
| Legendas | Obrigatório — SRT ou burn-in, facilita consumo em silent mode |
| Hospedagem sugerida | Wistia, Panda Video ou Vimeo (evitar YouTube pra VSL — anúncios quebram conversão) |
| URL | Inserir em `VSL_EMBED_URL` no arquivo `src/pages/TrialObrigado.tsx` após upload |
