  Yakut is a cross-platform command-line utility
  for diagnostics and management of Cyphal
  networks. It is designed for use either directly
  by humans or from automation scripts. Ask
  questions at https://forum.opencyphal.org

  Any long option can be provided via environment
  variable prefixed with `YAKUT_` such that an
  option `--foo-bar` for command `baz`, if not
  provided as a command-line argument, will be
  read from `YAKUT_BAZ_FOO_BAR`.
